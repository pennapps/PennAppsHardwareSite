To lasercut a part, save your solidworks as a .dwg file with the below parameters and send to laser@pennapps.com

- Use inches, not centimeters

- .dwg Coloring Guide:

  • Vector Cutting: RED
  • Vector Etching: YELLOW
  • Raster Etching: GREEN

Available Materials

  • MDF 1/4"
  • MDF 1/8"

Check out "Preparing Files for Laser Cutting" for more info and tips.

Tips on Exporting DWF files from Illustrator

  • CMYK Green - rastering
  • CMYK Red - .001” - cut
  • Avoid White Fills (but its forgivable) - they show up at dark red in solid works, and must be ignored.
  • NEVER “group” items. Ungroup everything before exporting as DWG.
  • There is an issue with trying to raster images and “live traced” images. They export as a PNG and the file path is hard coded in the DWG - Possible fix: “expand” the live trace.
